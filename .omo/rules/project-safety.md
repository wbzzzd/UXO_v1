# 项目安全规则

适用范围：UXO_v1 的所有 opencode、oh-my-openagent 和人工辅助开发。

## 默认安全姿态

- 默认使用本地模拟数据。
- 默认禁止真实设备控制、排弹动作、外部通信命令和数据库写入。
- 任何真实设备、真实通信或安全关键动作都必须先获得用户明确授权。

## 密钥

- 不得提交 `.env`、密钥、凭据或明文 provider key。
- `opencode.json` 必须使用环境变量引用。
- 如果发现明文 key，先脱敏配置，再提醒用户轮换密钥。

## 验证

- 提交或交接前检查 diff 中是否包含密钥形态。
- 不为通过验证而删除失败测试或弱化检查。
