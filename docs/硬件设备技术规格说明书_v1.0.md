## 1. 文档概述

### 1.1 目的

本文档定义排弹抢修指挥系统中所有硬件设备的功能定位、技术规格要求、通信接口标准及环境适应性指标。文档面向系统集成开发团队和后续设备采购环节，作为硬件选型、系统集成、出厂验收的技术依据。

### 1.2 适用范围

- **部署场景**：单机场场站标准配置
- **运行环境**：空军场站室内（指挥方舱/塔台机房）+ 室外（跑道、滑行道、停机坪及周边区域）
- **目标操作系统**：银河麒麟 V10 / 统信UOS V20（国产化要求）

### 1.3 术语说明

| 术语 | 说明 |
|------|------|
| UXO | UneXploded Ordnance，未爆弹药 |
| GPR | Ground Penetrating Radar，探地雷达 |
| LiDAR | Light Detection and Ranging，激光雷达 |
| MQTT | Message Queuing Telemetry Transport，物联网消息协议 |
| RTK | Real-Time Kinematic，实时动态差分定位 |
| IP防护等级 | Ingress Protection，防尘防水等级 |
| MTBF | Mean Time Between Failures，平均无故障工作时间 |

### 1.4 硬件系统整体边界

## 3. 计算与存储设备

### 3.1 CS-01 数据处理服务器

#### 3.1.1 功能定位

数据处理服务器是整个系统的计算核心，承担以下关键任务：
- 实时接收多传感器探测数据（激光点云、红外图像、可见光图像、雷达信号）
- 运行 AI 推理引擎，执行目标识别与分类（TensorFlow Lite 模型推理）
- 执行多源数据融合算法（时空对齐 → 特征提取 → 目标关联 → 状态估计）
- 运行路径规划与殉爆安全链计算
- 运行 MQTT Broker，作为全系统消息中枢
- 态势数据实时融合与推送

#### 3.1.2 性能指标要求

| 指标 | 要求 | 依据 |
|------|------|------|
| CPU | ≥16核 32线程，主频≥2.4GHz，x86-64架构 | 需同时处理8路传感器数据流 + AI推理 |
| GPU | ≥NVIDIA RTX 4000 Ada / A2000 级别，≥8GB显存，≥200 TOPS(INT8) | AI推理计算、点云处理加速 |
| 内存 | ≥128GB DDR4/DDR5 ECC | 实时点云缓存(≥100万点/秒) + 多路图像缓冲 |
| 系统盘 | ≥1TB NVMe SSD | 操作系统及应用部署 |
| 数据盘 | ≥4TB NVMe SSD (RAID1) | 实时数据缓冲及临时处理 |
| 网络 | 双口10GbE SFP+ + 双口1GbE RJ45 | 高带宽数据接入 + 管理网络 |
| 扩展槽 | ≥2个 PCIe 4.0 x16 | GPU及未来扩展 |

#### 3.1.3 软件环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | 银河麒麟 V10（服务器版）/ 统信UOS V20（服务器版） |
| 容器运行时 | Docker ≥20.10 或 Podman |
| AI运行时 | TensorFlow Lite ≥2.12，CUDA ≥11.8 |
| 消息中间件 | Eclipse Mosquitto ≥2.0（MQTT Broker） |
| 数据库客户端 | PostgreSQL 客户端库 ≥14 |

#### 3.1.4 通信接口

| 接口 | 用途 | 协议 |
|------|------|------|
| 10GbE SFP+ Port 1 | 连接核心交换机，传感器数据接入 | TCP/IP |
| 10GbE SFP+ Port 2 | 连接数据存储服务器，数据持久化 | TCP/IP |
| 1GbE RJ45 Port 1 | 管理网络 | TCP/IP |
| 1GbE RJ45 Port 2 | 备用/调试 | TCP/IP |
| MQTT (软件层) | 全系统消息总线，端口1883（内网） | MQTT v3.1.1 |

| 安装方式 | 19英寸标准机柜安装 |

### 3.3 CS-03 不间断电源（UPS）

#### 3.3.1 功能定位

为塔台机房内所有计算存储设备、网络设备提供不间断供电，确保在电网故障时系统能：
- 维持 ≥30分钟正常运行（覆盖柴油发电机启动时间）
- 安全执行数据持久化和系统关机流程
- 通过 SNMP/干接点向系统发送供电状态告警

#### 3.3.2 性能指标要求

| 指标 | 要求 |
|------|------|
| 额定容量 | ≥6kVA / 5.4kW（在线式） |
| 输入 | 交流220V±15%，50Hz±5% |
| 输出 | 纯正弦波，220V±2%，50Hz±0.5% |
| 后备时间 | ≥30分钟（满载） |
| 切换时间 | 0ms（在线式，无切换中断） |
| 通信接口 | SNMP网卡 + RS232串口 + 干接点 |
| 电池类型 | 免维护铅酸蓄电池或磷酸铁锂电池 |

#### 3.3.3 配置数量

| 配置 | 数量 |
|------|------|
| 每场站 | **1台**（或按负载配置 N+1 冗余） |

## 5. 探测传感器设备

### 5.1 DS-01 红外热成像系统

#### 5.1.1 功能定位

搭载于无人机光电吊舱内，通过探测目标与背景的温度差异，识别地表未爆弹药及其热特征，并支持沙尘环境下的穿尘成像。

#### 5.1.2 性能指标要求

| 指标 | 要求 | 依据 |
|------|------|------|
| 探测器类型 | 非制冷氧化钒(VOx)或制冷型InSb焦平面阵列 | 高灵敏度目标探测 |
| 分辨率 | ≥1280×1024 | SRS要求捕获精细热特征 |
