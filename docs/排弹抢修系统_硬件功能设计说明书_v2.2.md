# 排弹抢修系统 硬件功能设计说明书

**版本**：V2.3
**日期**：2025年7月
**范围**：功能需求定义，不定供应商、不定品牌型号
**依据**：排弹方案0919.docx、SRS、SDD、界面设计.docx

---

## 设计原则

| 原则 | 说明 |
|------|------|
| **只定义功能需求** | 定义各设备"必须做到什么"，不指定品牌型号。供应商选型是下一步工作 |
| **全国产化** | 所有设备须为中国自主研发生产，不依赖进口供应链 |
| **无人化智能化** | 全流程最小化人员进入危险区。探测在空中完成，处置由机器人执行，指挥员在方舱内远程操控 |
| **战场化部署** | 所有设备平时存仓库，战时拉出部署，跑道不留任何固定设施 |
| **封闭通信** | 不依赖公网，全系统在私有局域网/专有无线链路内运行 |
| **单载荷多次起降** | 每架无人机一次只搭载一种传感器，多次起降切换载荷 |
| **全程空中监视** | 一架专用监控无人机从系统就绪起持续在空，直至任务结束 |
| **货架产品优先** | 系统集成商不开发硬件，所有设备须为可采购的成熟产品 |

## 2. 作战流程

### 2.1 部署阶段（仓库 → 跑道，≤20分钟）

| 时间 | 动作 | 涉及设备 |
|------|------|---------|
| T+0 | 接到排弹命令 | — |
| T+0~5 | 两车编队从仓库驶至跑道南端部署点 | 运输车+指挥方舱车 |
| T+5~12 | 方舱开机、服务器启动、软件加载 | 服务器×2 + 工作站 + 交换机 |
| T+5~14 | 机器人卸下、上电自检、GPR天线安装、工具装载 | 排爆机器人 |
| T+8~18 | 无人机展开、起降坪铺设、电池装载、飞前检查 | 无人机×3 + 充电站 |
| T+18 | **监控无人机起飞**，60-80m悬停，回传全跑道画面 | 监控无人机 |
| T+20 | 全系统就绪，2架侦察无人机待命起飞 | — |

### 2.2 侦察扫描阶段

监控无人机T+18起飞后持续在空。侦察无人机分三轮飞行，每轮搭载不同传感器：

**第一轮：EO/IR扫描（主力，~30分钟）**

| 时间 | 动作 |
|------|------|
| T+20 | 侦察机A挂EO/IR、侦察机B挂EO/IR，同时起飞 |
| T+20~50 | 南北分区扫描，回传可见光+红外。服务器实时AI识别，标注地表可疑目标 |
| T+50~55 | 两机落地，换LiDAR吊舱+换电池 |

**第二轮：LiDAR扫描（补充，~30分钟）**

| 时间 | 动作 |
|------|------|
| T+55 | 两机挂LiDAR起飞 |
| T+55~85 | 南北分区扫描，回传三维点云。与第一轮数据融合，生成弹坑模型 |
| T+85~90 | 两机落地，换磁力仪+换电池 |

**第三轮：磁力扫描（地下铁磁目标，~30分钟）**

| 时间 | 动作 |
|------|------|
| T+90 | 两机挂磁力仪起飞，超低空飞行（≤3m，需地形跟随） |
| T+90~120 | 南北分区扫描，回传磁异常数据。检出地下铁磁目标 |
| T+120 | 两机落地。全跑道扫描完成 |

> **设计说明**：EO/IR排第一——它是主力传感器，覆盖了可见光+红外全部探测需求，指挥员可在最早时间开始确认目标。LiDAR第二——补充三维建模，有烟尘时承担部分探测。磁力仪第三——超低空飞行对飞行控制要求最高，需要前两轮积累的跑道模型数据。如果战况判断无烟尘且无地下目标需求，LiDAR和磁力扫描可根据实际情况裁减。

### 2.3 目标确认与排爆处置（无人化）

**机器人是唯一的执行平台**——同时承担GPR复察和排爆处置，无需独立探测载车。流程全无人化：

| 步骤 | 做什么 | 持续 |
|------|--------|------|
| **目标确认** | 指挥员在三维态势图上审核AI识别结果，生成优先级排序的目标处置队列 | 与扫描同步 |
| **自主导航至目标** | 机器人根据GPS坐标自主规划路径，行驶至目标点。全程由监控无人机俯拍监视 | 2~5min/目标 |
| **GPR精确复察** | 机器人放下底盘GPR天线贴地扫描，确认地下目标的精确深度、尺寸和位置。如为地表目标则跳过此步，直接用摄像头确认 | 2~3min/目标 |
| **处置执行** | 确认目标后，指挥员远程操控或机器人半自主选择工具执行排除。机械臂+末端工具组完成夹取/拆解/引爆 | 20~40min/目标 |
| **自主移至下一目标** | 处置完成后收拢工具和GPR天线，自主规划路径驶向下一个目标 | 2~5min |

> **无人化闭环**：从目标确认到处置完成，全程无需人员进入跑道。指挥员在方舱远程监督，紧急情况下可接管操控。机器人离线自主决策作为通信中断的冗余。

### 2.4 全程监控与通信中继

监控无人机T+18起飞后持续在空直至任务结束，角色三维一体：

| 角色 | 说明 |
|------|------|
| **全域态势监视** | 60-80m高度，变焦相机俯瞰全跑道。扫描阶段辅助发现大面积异常；处置阶段聚焦作业区 |
| **排爆作业俯拍** | 机器人处置时，变焦镜头对准作业点，提供高空俯拍补充机器人自身视角 |
| **空中通信中继** | 搭载Mesh中继载荷，作为空中一跳节点。对跑道全域通视，彻底解决地面设备NLOS通信 |

**热切换策略**：

- 监控无人机电量降至35%时，一架侦察无人机换上监控吊舱+中继载荷，起飞接替
- 接替机飞到同位置→指挥席确认画面→原机返航落地换电池
- 交接时5分钟双机同框，零间隙覆盖
- 电池配置：≥8块（3在用+3充电+2待命），4槽快充

### 2.5 效果评估

| 阶段 | 做什么 | 谁做 |
|------|--------|------|
| 处置后复察 | 每完成一个目标排爆，侦察机挂EO/IR飞抵处置点，对比处置前后图像 | 侦察无人机×1 |
| 任务结束 | 指挥席确认全部目标已排除，监控无人机降落，全系统撤收 | 全部 |

## 3. 空中平台功能需求

### 3.1 无人机平台（通用，3架同型）

**功能定位**：搭载各类吊舱执行侦察扫描、定点监控和通信中继任务。三架同型号，通过换吊舱切换角色，降低后勤复杂度。

| 需求项 | 需求内容 | 需求依据 |
|--------|---------|---------|
| **载荷能力** | ≥ 1.5kg（最重的磁力仪约1.6kg，需留余量） | 磁力仪是最大载荷 |
| **续航** | 载荷状态下 ≥ 35分钟 | 覆盖1500m×500m半区扫描 |
| **飞行速度** | ≥ 10m/s（扫描），≥ 20m/s（转场） | 扫描效率 |
| **超低空能力** | 支持≤3m超低空定高飞行，配地形跟随（激光/雷达高度计） | 磁力扫描硬需求 |
| **定位精度** | RTK差分，水平≤3cm、垂直≤5cm | 目标坐标精度 |
| **悬停稳定性** | 60-80m高度漂移 ≤ ±0.5m | 监控俯拍和通信中继角度稳定 |
| **工作温度** | -20°C ~ 50°C | 全季节全天候 |
| **防护等级** | ≥ IP54 | 沙尘、小雨 |
| **抗风** | ≥ 10m/s（5级） | 机场开阔地带 |
| **起降** | 垂直起降，可在便携起降坪或平整道面起降 | 无固定设施 |
| **换电** | 支持快拆换电，≤ 3分钟 | 接力策略 |
| **展开收纳** | 折叠入运输箱，展开 ≤ 5分钟 | 仓库存储、快速部署 |
| **国产化** | 全国产自主研发生产 | 军事应用硬性要求 |

**挂载接口**：单云台机械接口，供电 DC 12-24V ≥ 50W，千兆以太网数据，GNSS+PPS时间同步。

> **设计说明**：磁力扫描需要超低空飞行（≤3m），这是本轮新增的无人机能力要求。市场上已有的国产航磁系统（如北京桔灯UFO-CS铯光泵系统，适配国产无人机平台）证明这项能力可商业化实现。排弹方案0919中引用的无人机参数（15.8kg起飞、6kg载荷、1070mm轴距）与主流国产航磁解决方案吻合。

### 3.2 EO/IR光电吊舱（搭载于侦察无人机）

**功能定位**：一次飞行同时采集可见光图像和红外热图。这是成熟的商业化产品形态。

**市场验证**：EO/IR合一吊舱是货架产品，国产厂商可提供。红外分辨率从640×512到1280×1024均有，可按需选型。

| 需求项 | 需求内容 | 依据 |

| 可见光相机 | ≥ 2000万像素，4K视频，光学变焦 ≥ 10× | AI形态识别和人工判读 |
| 红外热像仪 | 分辨率 ≥ 640×512，NETD ≤ 50mK | 热异常探测 |
| 视频输出 | 两路独立RTSP（可见光+红外），H.264/H.265 | |
| 云台 | 三轴机械增稳 | |
| 重量 | ≤ 1.2kg | |
| 功耗 | ≤ 30W | |
| 工作温度 | -20°C ~ 50°C | |
| 防护等级 | ≥ IP54 | |
| 挂载接口 | 单云台机械接口 + 单电缆（供电+以太网） | |
| 国产化 | 全国产 | |

### 3.3 LiDAR吊舱（搭载于侦察无人机，优先级低于EO/IR）

**功能定位**：获取跑道表面三维点云。主要价值在穿烟/穿尘能力——遭受空袭后可能出现的扬尘环境中，被动成像（可见光/红外）可能失效，LiDAR是可靠的补充手段。无烟尘时可跳过本轮。

| 需求项 | 需求内容 |
|--------|---------|
| 点频 | ≥ 100,000 点/秒 |
| 测距精度 | ≤ 5cm @ 100m |
| 测距范围 | ≥ 150m @ 20%反射率 |
| 回波 | 支持多回波（≥2次），穿透扬尘/轻烟 |
| 激光等级 | Class 1人眼安全 |
| 重量 | ≤ 1.5kg |
| 功耗 | ≤ 40W |
| 工作温度 | -20°C ~ 50°C |
| 防护等级 | ≥ IP54 |
| 挂载接口 | 单云台机械接口 + 单电缆 |
| 国产化 | 全国产 |

### 3.4 机载磁力仪系统（搭载于侦察无人机）

**功能定位**：超低空飞行（≤3m），通过测量地磁场局部异常探测地下铁磁金属目标。作为EO/IR和LiDAR的互补——两者主要覆盖地表，磁力仪探测地下。

**市场验证**：国产航磁系统是成熟产品。北京桔灯UFO-CS铯光泵系统2017年商业化，已飞行25万公里，明确支持UXO/地雷探测，配套国产数据处理软件（Archon + GeoProbe UAV）。磁力仪本身不绑定特定无人机平台。

| 需求项 | 需求内容 |
|--------|---------|
| 传感器类型 | 铯光泵或同等精度磁力仪 |
| 灵敏度 | ≤ 0.1 nT（或 ≤ 50 pT/√Hz @ 0.1-100Hz） |
| 采样率 | ≥ 50 Hz |
| 重量 | ≤ 2kg（含电池和安装支架） |
| 飞行高度 | ≤ 3m（越低越好，≤1m 为优） |
| 地形跟随 | 必须配备激光/雷达高度计，保证超低空稳定飞行 |
| 安装方式 | 刚性支架快拆，≤ 2分钟装卸 |
| 日变校正 | 配地面日变站（磁力基站），同步记录地磁场日变化 |
| 数据处理 | 配套国产航磁数据处理软件（补偿、滤波、日变改正、目标反演定位） |
| 输出 | 磁异常分布图 + 目标GPS坐标列表 |
| 功耗 | ≤ 30W |
| 工作温度 | -20°C ~ 50°C |
| 国产化 | 全国产，自主知识产权 |

> **为什么磁力仪上无人机而非载车**：机载磁力扫描效率是地面人工的5-10倍（1公顷/小时 vs 1-2公顷/天）；无人机超低空飞行可达0.2-1m离地高度，检出率远高于地面设备的1-2m传感器高度；最重要的是——操作员不在危险区域。国产方案已完全成熟。

### 3.6 通信中继载荷（搭载于监控无人机）

**功能定位**：将监控无人机变为空中Mesh网络节点。60-80m高度对跑道全域通视，地面设备的NLOS问题（弹坑遮挡、建筑物遮挡）通过空中一跳直达解决。消除对地面中继节点的依赖。

| 需求项 | 需求内容 |

### 4.2 链路需求

| 链路 | 端点 | 数据 | 带宽 | 延迟 | 距离 | 介质 |
|------|------|------|------|------|------|------|
| L1 | 服务器↔交换机 | 全系统数据 | ≥1Gbps | ≤1ms | ≤10m | 网线 |
| L2 | 交换机↔指挥席 | 态势+视频 | ≥100Mbps | ≤50ms | ≤200m | 光纤 |
| L3 | 无人机↔地面站 | 视频+传感器数据 | ≥20Mbps/架 | ≤200ms | ≥5km | 私有图传 |
| L4 | 地面站↔服务器 | 汇聚数据 | ≥100Mbps | ≤10ms | ≤200m | 网线 |
| L5 | 机器人↔空中中继↔Mesh基站 | 视频+遥测+GPR数据 | ≥10Mbps | ≤100ms | ≥3km | Mesh |
| L6 | 操作台↔机器人 | 有线精操作 | ≥100Mbps | ≤5ms | ≥100m | 光纤 |

### 4.3 空中中继的战术价值

60-80m高度对3000m×500m跑道**全区域通视**。机器人无论进入弹坑、跑到跑道边缘、被建筑物遮挡，空中中继都能维持链路。地面中继从"必需"降为"备选"（仅空中中继不可用时的降级方案）。

### 4.4 频率规划

| 频段 | 用途 | 原因 |
|------|------|------|
| ≥ 5GHz | 无人机图传（3机各独立信道） | 高带宽低延迟 |
| ≤ 1GHz | Mesh自组网（机器人+空中中继+基站）| 穿障能力强 |
| 有线光纤 | 精操作和室内骨干 | 零延迟零干扰 |

### 4.5 通信容灾

| 场景 | 设备行为 |
|------|---------|
| 无人机图传中断 | 自动返航，机载存储保留数据 |
| 空中中继失效 | Mesh网络回落直连或地面中继 |
| 全网断 | 机器人停止待命，30秒无恢复沿原路退回；可切换有线精操 |
| 机器人断网 | 机器人本机存储GPR数据，恢复后同步 |
| 服务器宕机 | 地面站本地缓存；机器人本机简版决策 |

## 5. 排爆机器人（含集成探地雷达）

### 5.1 设计理念

排爆机器人是本系统唯一的地面执行平台。基于"无人化智能化"原则，取消独立探测载车，将探地雷达（GPR）直接集成于机器人——到达目标点后自行GPR确认，确认后自行排爆。一台机器人完成从"确认"到"排除"的全闭环，全程无需人员进入跑道。

### 5.2 平台基础需求

| 需求项 | 需求内容 | 
|--------|---------|
| 重量 | ≤ 80kg（含电池、GPR天线和工具组）|
| 尺寸 | ≤ 1200×800×800mm（收纳状态）|
| 速度 | ≥ 2m/s，无级调速 |
| 越障 | 垂直≥300mm，越沟≥300mm，爬坡≥30°（GPR天线收起状态）|
| 连续工作 | ≥ 3小时 |
| 防护等级 | ≥ IP66（底盘），≥ IP65（机械臂），≥ IP65（GPR天线）|
| 工作温度 | -20°C ~ 50°C |
| 自主导航 | GNSS+RTK差分定位 + 惯性导航 + 视觉SLAM辅助，自主规划路径行驶至目标坐标 |
| 障碍检测 | 前置避障传感器（超声波/毫米波），自动绕行 |
| 离线决策 | 通信中断时本机执行预设处置预案，任务完成后自主返航 |
| 国产化 | 全国产 |

### 5.3 GPR子系统（集成于机器人）

**安装方式**：机器人底盘前部下方设置GPR安装位。天线通过可收放机构连接——行驶时收起（不影响通过性），到达目标点后放下（贴地或距地面≤15cm进行扫描）。

**工作流程**：机器人自主行驶至目标GPS坐标 → 放下GPR天线 → 近场扫描（1m×1m区域，~30秒）→ 回传SEG-Y数据至服务器 → 服务器AI判断：确认弹药存在/不存在/不确定 → 确认则进入处置流程，不确定则人工介入判断。

| 需求项 | 需求内容 | 依据 |
|--------|---------|------|
| 中心频率 | ≥ 800MHz | 兼顾分辨率和穿透深度 |
| 探测深度 | ≥ 1.5m | 战斗部典型穿透深度 |
| 天线形态 | 小型化天线（非大车阵列），适合300mm×500mm安装空间 | 机器人底盘尺寸限制 |
| 扫描范围 | 定点扫描≥1m²，分辨率≤5cm | 精确定位弹药边界 |
| 输出 | SEG-Y标准格式，含GPS坐标 | 排弹方案指定 |
| 天线重量 | ≤ 8kg | 机器人载荷预算 |
| 收放机构 | 电动升降/折叠，行程≥200mm，≤5秒完成状态切换 | |
| 防护等级 | ≥ IP65 | 贴地工作时防尘防水 |
| 数据接口 | 千兆以太网，连接机器人主控 | |
| 工作温度 | -20°C ~ 50°C | |
| 国产化 | 全国产 |

> **设计说明**：机载GPR不成熟（检出率31%），大型车载GPR需要有人驾驶。将小型化GPR集成在排爆机器人上，利用机器人自主导航至目标点作定点扫描，既避免了机载GPR的性能问题，又消除了人为驾驶的需要，符合无人化原则。GPR天线的可收放设计解决了越野通过性（行驶时收起）和探测精度（工作时贴地）的矛盾。

### 5.8 机器人整备与操作流程

```
部署阶段
  T+5~14  机器人从运输车卸下 → 上电自检 → 安装GPR天线 → 装载工具组 → 通信链路建立
          工具架预装4种工具，GPR天线手动安装到机器人底盘安装位

作战阶段
  目标确认后 → 机器人收到目标队列 (GPS坐标 + 类型 + 优先级)
  For each 目标:
    1. 自主导航行驶至目标GPS坐标     (2~5min)
    2. 放下GPR天线，定点扫描确认      (2~3min，仅地下目标)
    3. AI推荐工具 → 指挥员确认 → 执行 (20~40min)
    4. 监控无人机复察确认处置效果
    5. 收起GPR天线和工具 → 驶向下个目标
```

## 附录：设计决策记录

| 编号 | 决策 | 理由 |
|------|------|------|
| D-001 | 全国产化 | 军事应用硬性要求 |
| D-002 | 无人化智能化 | 全流程最小化人员进入危险区。探测在空中完成，处置由机器人执行 |
| D-003 | 3架同型无人机 | 通过换吊舱切换角色，降低后勤复杂度 |
| D-004 | 单载荷多次起降 | 三合一吊舱不存在；系统集成商不开发硬件；每种传感器独立采购 |
| D-005 | EO/IR第一、LiDAR第二、磁力第三 | EO/IR覆盖全部地表探测；LiDAR补充三维和穿尘；磁力补充地下 |
| D-006 | 磁力仪机载 | 机载效率5-10倍于地面；国产航磁系统成熟；排弹方案本身即按机载设计 |
| D-007 | GPR集成于排爆机器人 | 取消探测载车。机器人到达目标点后自行GPR确认，确认后自行排爆。一台设备闭环，真正无人化 |
| D-008 | 监控无人机全程在空+空中中继 | 高空对跑道全域通视，消除NLOS；同时提供态势监视和通信中继 |
| D-009 | 双机热切换 | 单机续航不足以覆盖全任务；轮换保证零间隙监控 |
| D-010 | 全封闭通信 | 战时不可依赖公网；私有图传+Mesh+有线三层架构 |
| D-011 | 不定品牌型号 | 功能设计先行，选型采购是下一步工作 |
