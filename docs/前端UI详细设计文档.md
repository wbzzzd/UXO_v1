## 1. 通用设计规范

### 1.1 设计原则

- **一致性原则**: 所有窗口和组件使用统一的颜色、字体、控件风格
- **响应性原则**: 界面操作需提供即时反馈，包括按钮状态变化、加载动画、提示信息
- **可访问性原则**: 支持键盘快捷键操作，按钮尺寸满足触屏操作要求（最小44×44像素）
- **模块化原则**: UI组件设计为可复用，如列表项、卡片、弹窗等

### 1.2 开发要求

本文档实现须满足以下开发要求：

1. **模拟数据源**: 所有展示数据均使用模拟数据源，不依赖真实后端接口
2. **模拟图片**: 所有图片素材可以使用占位图或程序生成的模拟图片
3. **展示效果优先**: 开发完成后系统须可正常运行并展示完整效果
4. **可操作性**: 前端须可上手操作，所有交互功能均须可实际运行

## 2. 系统主界面设计

### 2.1 整体布局结构

系统主界面是指挥席的核心操作界面，采用经典的三栏式布局，从上到下依次排列为：

1. **标题栏**：高度30px，左侧显示窗口图标和程序名称，右侧为标准窗口控制按钮
2. **菜单栏**：高度30px，包含文件、视图、工具、设备、帮助等菜单
3. **工具栏**：高度48px，包含图层控制、测量工具、坐标拾取、视角复位、同步状态、书签、打开设备控制台等快捷按钮
4. **内容区**：
   - 左侧面板：宽度320px，可折叠，包含目标、任务、设备列表的标签切换
   - 三维视图区：自适应宽度，显示数字孪生地图
   - 右侧面板：宽度360px，可折叠，显示目标详情或任务规划等上下文内容
5. **状态栏**：高度60px，显示设备状态、告警滚动条和紧急停止按钮

### 2.2 标题栏详细设计

#### 2.2.1 布局

- **左侧区域**：显示程序图标和标题文本"排弹抢修系统指挥席 V1.0"
- **右侧区域**：标准窗口控制按钮（最小化、最大化/还原、关闭）

#### 2.2.2 功能说明

- **双击标题栏**：窗口最大化/还原
- **拖拽标题栏**：移动窗口位置
- **点击最小化**：窗口最小化到任务栏
- **点击最大化**：窗口全屏显示
- **点击关闭**：触发关闭确认（有关键任务时）

### 2.3 菜单栏详细设计

| 保存方案 | Ctrl+S | 将当前任务规划保存为.plan文件               |

#### 2.3.2 视图菜单

| 显示右侧面板      | 切换右侧面板可见性         |
| 显示状态栏       | 切换状态栏可见性          |
| 视角 → 顶视     | 切换到俯视视角           |
| 视角 → 侧视     | 切换到侧视视角           |
| 视角 → 跟随选中设备 | 跟随选中设备视角（设备选中时可用） |

#### 2.3.3 工具菜单

| ---- | ------------------- |

| 数据同步 | 手动触发同步离线数据库，状态栏显示进度 |

### 2.4 工具栏详细设计

工具栏位于菜单栏下方，固定高度48px，背景色#2D2D2D。

#### 2.4.1 控件排列（从左到右）

1. **图层控制下拉按钮**
   - 点击展开下拉面板，包含复选框列表
   - 复选框选项：目标标注、设备模型、路径轨迹、作业区域、禁区、红外热像叠加、探地雷达数据
   - 选择/取消选择后，三维视图立即更新
2. **测量工具下拉按钮**
   - 点击展开工具选项：距离测量、面积测量、坐标拾取
   - 选中任一工具后，鼠标进入测量模式，状态栏显示提示"点击地图开始测量"
   - 再次点击该工具可退出测量模式
3. **坐标拾取按钮**
   - 点击后鼠标进入坐标拾取模式
   - 点击地图任意位置，在点击处显示浮窗，显示该点的经纬度和本地坐标
   - 浮窗停留3秒后自动消失
4. **视角复位按钮**
   - 点击后相机平滑过渡到初始俯瞰视角
   - 覆盖整个机场区域
5. **同步状态按钮**
   - 点击后图标旋转动画，表示正在刷新数据
   - 前端向后端请求最新设备状态、目标列表等，更新UI
6. **书签下拉按钮**
   - 下拉菜单包含："保存当前视角"和已保存书签列表
   - 点击"保存当前视角"弹出输入对话框，输入书签名后保存当前相机位置
   - 点击已保存书签项，相机平滑切换至对应视角
   - 书签数据本地存储
7. **打开设备控制台按钮**
   - 点击后弹出设备选择对话框（模态）
   - 对话框列出所有在线设备
   - 支持双击设备或选中后点击"打开"按钮
   - 若该设备控制台已打开，则激活已有窗口

## 3. 左侧面板设计

左侧面板宽度固定320px，背景色#252526，可折叠。

### 3.1 标签页切换设计

面板顶部有三个标签并排：目标、任务、设备。当前选中标签有下划线或高亮背景（#2A3F54）。点击标签切换下方列表内容，同时更新右侧面板上下文。

### 3.2 目标列表页设计

#### 3.2.1 状态子标签

目标列表页在列表上方有三个子标签（默认选中"待处置"）：待处置、处置中、已完成。

#### 3.2.2 顶部工具栏

包含以下控件：

1. **筛选按钮**（漏斗图标）
   - 点击展开下拉菜单
   - 筛选选项：
     - 目标类型：航弹、子母弹、地雷、其他（支持多选）
     - 威胁等级：高、中、低（支持多选）
2. **搜索框**
   - 输入文本时实时过滤
   - 按编号、类型、描述进行模糊匹配
   - 右侧带清除按钮
3. **刷新按钮**
   - 重新从后端请求目标列表

#### 3.2.3 列表项设计

每行高度56px，包含以下元素（从左到右）：

1. **复选框**（宽30px）：用于多选操作
2. **威胁等级色条**（宽4px）：红色-高威胁、橙色-中威胁、黄色-低威胁
3. **类型图标**（24×24px）：根据目标类型显示对应图标
4. **文本区域**：
   - 第一行：类型 + 型号（14px白色）
   - 第二行：置信度 + 简要位置（12px灰色#AAA）

#### 3.2.4 列表交互

- **选中行**：背景色高亮#2A3F54
- **悬停行**：背景色微亮#2A2A2A
- **虚拟滚动**：使用虚拟滚动技术优化长列表性能

#### 3.2.5 批量操作栏

当至少一个复选框被选中时，列表底部出现批量操作栏：

- 高度：48px
- 背景色：#333333
- 包含两个按钮：
  - "分配任务"：点击后右侧面板切换为任务规划面板，并预填选中目标ID
  - "标记忽略"：弹出确认框，确认后向后端发送忽略请求

#### 3.2.6 右键菜单

在列表项上右键时出现，菜单项包括：查看详情、创建排爆任务、标记为误报、忽略（仅本次会话）。

### 3.3 任务列表页设计

#### 3.3.1 卡片列表布局

- 采用卡片列表形式
- 每个卡片高度：80px
- 卡片间距：2px

#### 3.3.2 卡片内容设计

每个卡片包含：

1. **左侧优先级色条**：P0-红色、P1-橙色、P2-黄色
2. **任务编号**：显示任务唯一编号
3. **执行设备名称**：显示分配的设备名称
4. **进度条**：水平进度条显示百分比，填充色#4A7A4C
5. **状态标签**：不同状态使用不同背景色（规划中、待审批、执行中、已完成、失败）

#### 3.3.3 卡片交互

- **单击卡片**：右侧面板切换为任务规划面板，加载该任务数据
- **双击卡片**：三维视图聚焦到该任务关联的第一个目标位置
- **右键菜单**：查看报告（仅已完成任务）、终止任务（仅执行中任务）、删除任务（仅草稿状态）

### 3.4 设备列表页设计

#### 3.4.1 列表项设计

每行高度64px，包含以下元素（从左到右）：

1. **设备图标**（24×24px）：根据设备类型显示对应图标
2. **设备名称/编号**：显示设备名称和编号
3. **状态文本**：在线-绿色、离线-灰色、任务中-黄色、故障-红色
4. **电量图标 + 百分比**：
   - 电池图标填充色：绿色>60%、黄色20%\~60%、红色<20%

#### 3.4.2 列表交互

- **单击设备**：选中设备，三维视图聚焦到设备位置，右侧面板显示设备基本信息并提供"打开控制台"按钮
- **双击设备**：直接打开该设备的控制台窗口
- **右键菜单**：打开控制台、查看设备信息、重启设备

## 4. 三维视图区设计

三维视图区是主窗口的核心区域，用于展示数字孪生地图和实时态势。

### 4.1 渲染内容规范

#### 4.1.1 基础地形

- 内容：从本地加载的机场三维模型（跑道、滑行道、停机坪、建筑、地形高程）
- 模型格式：.gltf或.obj
- 技术：采用LOD技术分级加载

#### 4.1.2 动态标注

1. **目标标注**
   - 技术：公告牌技术
   - 样式：圆形底座加类型符号
   - 颜色：根据威胁等级（红色#FF5252、橙色#FFB74D、黄色#FFF176）
   - 附加信息：图标下方可显示目标ID或类型缩写
2. **设备标注**
   - 样式：精细3D模型
   - 状态光环：周围悬浮状态光环（圆环，颜色对应状态）
   - 动画：光环有呼吸动画
3. **路径标注**
   - 样式：虚线（白色，实线长10px间隔10px）
   - 路径点：显示为小圆球（直径8px，可拖拽）
   - 选中路径点：高亮为黄色
4. **作业区域**
   - 样式：半透明平面
   - 颜色：绿色-应急起降带、蓝色-作业区、红色-禁区
   - 边框：边缘有实线边框

#### 4.1.3 图层叠加

1. **红外热像叠加**
   - 样式：半透明热图形式（红热模式）
   - 控制：通过工具栏图层控制开关
2. **探地雷达数据**
   - 样式：地下剖面图或点云形式
   - 交互：可调节透明度

### 4.2 交互操作规范

#### 4.2.1 相机控制

| 操作方式 | 功能          |
| ---- | ----------- |
| 左键拖拽 | 旋转视角（绕场景中心） |
| 右键拖拽 | 平移视角        |
| 滚轮   | 缩放          |
| W    | 向前平移        |
| A    | 向左平移        |
| S    | 向后平移        |
| D    | 向右平移        |
| Q    | 垂直下降        |
| E    | 垂直上升        |

#### 4.2.2 对象选择

1. **单击目标/设备**
   - 触发射线检测
   - 选中后对象周围出现高亮轮廓（发光边缘）
   - 右侧面板更新为对应详情
2. **双击目标**
   - 平滑插值相机至目标位置
   - 距离目标约5米
   - 视角俯视
3. **双击设备**
   - 打开设备控制台窗口
4. **框选（Ctrl+左键拖拽）**
   - 绘制矩形选框
   - 释放后高亮框内所有目标
   - 右侧面板显示批量操作

#### 4.2.3 悬浮信息

- 鼠标悬停在目标/设备上超过500ms
- 显示工具提示：
  - 目标：类型、置信度
  - 设备：名称、状态、电量

#### 4.2.4 右键菜单（上下文相关）

1. **在目标上右键**：查看详情、创建排爆任务、标记误报、忽略
2. **在设备上右键**：打开控制台、查看信息、重启设备
3. **在空白处右键**：新建任务、测量、视角书签

- 确认对话框包含：标题、详细说明、"确认"按钮（红色警示按钮）、"取消"按钮

### 12.3 错误提示规范

- 操作失败时显示Toast提示
- 包含错误图标和错误信息
- 严重错误显示模态对话框

### 12.4 成功提示规范

- 操作成功时显示Toast提示
- 包含成功图标和成功信息
- 自动消失

### 12.5 键盘操作规范

- 所有可点击元素支持Tab键聚焦
- Enter键触发默认操作
- Esc键取消/关闭
- 方向键导航列表
