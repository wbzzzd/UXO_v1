## 1. 通用设计规范

### 1.1 设计原则

- **一致性原则**: 所有窗口和组件使用统一的颜色、字体、控件风格
- **响应性原则**: 界面操作需提供即时反馈，包括按钮状态变化、加载动画、提示信息
- **可访问性原则**: 支持键盘快捷键操作，按钮尺寸满足触屏操作要求（最小44×44像素）
- **模块化原则**: UI组件设计为可复用，如列表项、卡片、弹窗等

### 1.2 开发要求

本文档实现须满足以下开发要求：

1. **模拟数据源**: 所有展示数据均使用模拟数据源，不依赖真实后端接口
2. **模拟图片**: 所有图片素材可以使用占位图或程序生成的模拟图片
3. **展示效果优先**: 开发完成后系统须可正常运行并展示完整效果
4. **可操作性**: 前端须可上手操作，所有交互功能均须可实际运行

## 2. 系统主界面设计

### 2.1 整体布局结构

系统主界面是指挥席的核心操作界面，采用高密度仪表盘式布局，从上到下依次排列为：

1. **标题栏**：高度32px，左侧显示系统名称和程序版本，右侧显示用户名、时间和在线状态
2. **内容区**：
   - 左侧导航栏：宽度80px，包含态势、探测、决策、设备、统计、配置等导航菜单
   - 中间主视图区域：占60%宽度，包含视频流分屏区和信息面板
   - 右侧面板：占30%宽度，包含三维态势地图、设备状态、决策建议
3. **状态栏**：高度28px，显示在线设备数、目标数、告警数、系统资源、网络状态

## 3. 左侧面板设计

左侧面板宽度固定320px，背景色#252526，可折叠。

### 3.1 标签页切换设计

面板顶部有三个标签并排：目标、任务、设备。当前选中标签有下划线或高亮背景（#2A3F54）。点击标签切换下方列表内容，同时更新右侧面板上下文。

### 3.2 目标列表页设计

#### 3.2.1 状态子标签

目标列表页在列表上方有三个子标签（默认选中"待处置"）：待处置、处置中、已完成。

#### 3.2.2 顶部工具栏

包含以下控件：

1. **筛选按钮**（漏斗图标）
   - 点击展开下拉菜单
   - 筛选选项：
     - 目标类型：航弹、子母弹、地雷、其他（支持多选）
     - 威胁等级：高、中、低（支持多选）
2. **搜索框**
   - 输入文本时实时过滤
   - 按编号、类型、描述进行模糊匹配
   - 右侧带清除按钮
3. **刷新按钮**
   - 重新从后端请求目标列表

#### 3.2.3 列表项设计

每行高度56px，包含以下元素（从左到右）：

1. **复选框**（宽30px）：用于多选操作
2. **威胁等级色条**（宽4px）：红色-高威胁、橙色-中威胁、黄色-低威胁
3. **类型图标**（24×24px）：根据目标类型显示对应图标
4. **文本区域**：
   - 第一行：类型 + 型号（14px白色）
   - 第二行：置信度 + 简要位置（12px灰色#AAA）

#### 3.2.4 列表交互

- **选中行**：背景色高亮#2A3F54
- **悬停行**：背景色微亮#2A2A2A
- **虚拟滚动**：使用虚拟滚动技术优化长列表性能

#### 3.2.5 批量操作栏

当至少一个复选框被选中时，列表底部出现批量操作栏：

- 高度：48px
- 背景色：#333333
- 包含两个按钮：
  - "分配任务"：点击后右侧面板切换为任务规划面板，并预填选中目标ID
  - "标记忽略"：弹出确认框，确认后向后端发送忽略请求

#### 3.2.6 右键菜单

在列表项上右键时出现，菜单项包括：查看详情、创建排爆任务、标记为误报、忽略（仅本次会话）。

### 3.3 任务列表页设计

#### 3.3.1 卡片列表布局

- 采用卡片列表形式
- 每个卡片高度：80px
- 卡片间距：2px

#### 3.3.2 卡片内容设计

每个卡片包含：

1. **左侧优先级色条**：P0-红色、P1-橙色、P2-黄色
2. **任务编号**：显示任务唯一编号
3. **执行设备名称**：显示分配的设备名称
4. **进度条**：水平进度条显示百分比，填充色#4A7A4C
5. **状态标签**：不同状态使用不同背景色（规划中、待审批、执行中、已完成、失败）

#### 3.3.3 卡片交互

- **单击卡片**：右侧面板切换为任务规划面板，加载该任务数据
- **双击卡片**：三维视图聚焦到该任务关联的第一个目标位置
- **右键菜单**：查看报告（仅已完成任务）、终止任务（仅执行中任务）、删除任务（仅草稿状态）

### 3.4 设备列表页设计

#### 3.4.1 列表项设计

每行高度64px，包含以下元素（从左到右）：

1. **设备图标**（24×24px）：根据设备类型显示对应图标
2. **设备名称/编号**：显示设备名称和编号
3. **状态文本**：在线-绿色、离线-灰色、任务中-黄色、故障-红色
4. **电量图标 + 百分比**：
   - 电池图标填充色：绿色>60%、黄色20%\~60%、红色<20%

#### 3.4.2 列表交互

- **单击设备**：选中设备，三维视图聚焦到设备位置，右侧面板显示设备基本信息并提供"打开控制台"按钮
- **双击设备**：直接打开该设备的控制台窗口
- **右键菜单**：打开控制台、查看设备信息、重启设备

## 4. 三维态势地图设计

三维态势地图位于右侧面板顶部，是主视图的重要组成部分，用于展示数字孪生地图和实时态势。

### 4.1 渲染区域

三维地图占据右侧面板的主要空间，默认为窄条幅显示（约占右侧面板60%高度），支持缩放和切换显示模式。

### 4.1 渲染内容规范

#### 4.1.1 基础地形

- 内容：从本地加载的机场三维模型（跑道、滑行道、停机坪、建筑、地形高程）
- 模型格式：.gltf或.obj
- 技术：采用LOD技术分级加载

#### 4.1.2 动态标注

1. **目标标注**
   - 技术：公告牌技术
   - 样式：圆形底座加类型符号
   - 颜色：根据威胁等级（红色#FF5252、橙色#FFB74D、黄色#FFF176）
   - 附加信息：图标下方可显示目标ID或类型缩写
2. **设备标注**
   - 样式：精细3D模型
   - 状态光环：周围悬浮状态光环（圆环，颜色对应状态）
   - 动画：光环有呼吸动画
3. **路径标注**
   - 样式：虚线（白色，实线长10px间隔10px）
   - 路径点：显示为小圆球（直径8px，可拖拽）
   - 选中路径点：高亮为黄色
4. **作业区域**
   - 样式：半透明平面
   - 颜色：绿色-应急起降带、蓝色-作业区、红色-禁区
   - 边框：边缘有实线边框

#### 4.1.3 图层叠加

1. **红外热像叠加**
   - 样式：半透明热图形式（红热模式）
   - 控制：通过工具栏图层控制开关
2. **探地雷达数据**
   - 样式：地下剖面图或点云形式
   - 交互：可调节透明度

### 4.2 交互操作规范

#### 4.2.1 相机控制

| 操作方式 | 功能          |
| ---- | ----------- |
| 左键拖拽 | 旋转视角（绕场景中心） |
| 右键拖拽 | 平移视角        |
| 滚轮   | 缩放          |
| W    | 向前平移        |
| A    | 向左平移        |
| S    | 向后平移        |
| D    | 向右平移        |
| Q    | 垂直下降        |
| E    | 垂直上升        |

#### 4.2.2 对象选择

1. **单击目标/设备**
   - 触发射线检测
   - 选中后对象周围出现高亮轮廓（发光边缘）
   - 右侧面板更新为对应详情
2. **双击目标**
   - 平滑插值相机至目标位置
   - 距离目标约5米
   - 视角俯视
3. **双击设备**
   - 打开设备控制台窗口
4. **框选（Ctrl+左键拖拽）**
   - 绘制矩形选框
   - 释放后高亮框内所有目标
   - 右侧面板显示批量操作

#### 4.2.3 悬浮信息

- 鼠标悬停在目标/设备上超过500ms
- 显示工具提示：
  - 目标：类型、置信度
  - 设备：名称、状态、电量

#### 4.2.4 右键菜单（上下文相关）

1. **在目标上右键**：查看详情、创建排爆任务、标记误报、忽略
2. **在设备上右键**：打开控制台、查看信息、重启设备
3. **在空白处右键**：新建任务、测量、视角书签

### 4.3 悬浮工具栏设计

#### 4.3.1 位置和样式

- 位置：左上角
- 背景：半透明（#1E1E1E 80%）
- 交互：鼠标移入展开，移出收缩（或常驻）

#### 4.3.2 包含控件

- 图层快捷按钮
- 测量工具快捷按钮
- 视角书签下拉列表

- 高度：120px
- 背景色：#252526
- 内边距：8px

#### 7.6.1 预定义任务命令

**基础控制组**：

- "待命"：无人机保持当前位置悬停，等待后续指令

- "降落"：在当前降落点降落
- "返航"：返回起飞点并降落

**任务执行组**：

- "执行侦察"：按预设航线执行区域侦察任务（需预先规划航线）
- "执行排爆"：若无人机搭载排爆设备，执行未爆弹处置任务（需指定目标）
- "拍照"：拍摄当前画面并回传照片
- "录像"：开始/停止录像（切换按钮，开始录像时变为红色）

**紧急操作组**：

- "紧急返航"：红色按钮，立即中断当前任务返航（需二次确认）
- "悬停保持"：紧急悬停

#### 7.6.2 自定义命令输入区

- 位置：底部右侧
- 包含：命令输入框 + "发送"按钮
- 功能：允许操作员输入自定义指令（需遵循后端协议），用于扩展

#### 7.6.3 命令按钮交互

- 按钮显示"发送中"状态（加载动画）
