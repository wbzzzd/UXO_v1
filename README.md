# 排弹抢修指挥系统

## 项目简介

本项目为排弹抢修指挥系统软件，旨在为空军场站提供智能化的排爆管理软件。系统具备全域探测、目标识别、三维态势呈现、智能决策和排爆处置等功能。

## 技术架构

- **开发框架**: Qt 5.15 LTS
- **3D渲染**: Qt 3D
- **开发环境**: Ubuntu 20.04 LTS
- **目标部署**: 银河麒麟 V10 / 统信UOS V20
